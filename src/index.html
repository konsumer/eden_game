Game Engine Ideas

Demos

2D sidescroller

This is a 2D side-scroller engine. It uses pixi.js 2D engine & animations are made with spine. It demonstrates parallax layer-based scrolling, and animation.

3D terrain

This is a 3D world generated in shader, using BabylonJS 3D engine. The heightmap (image) defines which textures are loaded, and all textures are blended, based on height.n This also demonstrates a simple skybox & water shader.

Build System

I am using grunt with node-webkit to build standalone lin/win/mac versions of this demo.
